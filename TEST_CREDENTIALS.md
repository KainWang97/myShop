# 測試帳號文件 (Test Credentials)

本系統目前使用 Mock 資料模擬登入功能，需要正確的帳號密碼才能登入。

---

## 管理員帳號 (Admin)

擁有完整權限，可進入 Dashboard 管理商品、查看所有訂單與訊息。

| 項目     | 值                   |
| -------- | -------------------- |
| Email    | `admin@komorebi.com` |
| Password | `admin`              |
| 權限     | ADMIN                |

**功能權限**:

- 商品管理 (CRUD)
- 商品規格管理 (顏色/尺寸/庫存)
- 分類管理
- 訂單狀態更新
- 訊息回覆

---

## 一般會員帳號 (Member)

一般會員權限，可瀏覽商品、購物及查看自己的訂單。

| 項目     | 值              |
| -------- | --------------- |
| Email    | `test@test.com` |
| Password | `test123`       |
| 權限     | MEMBER          |

**功能權限**:

- 瀏覽商品
- 選擇規格 (顏色/尺寸)
- 加入購物車
- 結帳 (支援匯款與超商取貨)
- 訂單付款備註 (匯款資訊)
- 查看個人訂單

---

## 註冊新帳號

在登入畫面點擊「還沒有帳號？立即註冊」可以註冊新會員。

- 新註冊的帳號權限為 `MEMBER`
- 密碼至少需要 4 個字元
- Email 不可與現有帳號重複

> **注意**: Mock 模式下，新註冊的帳號資料僅存在於當前 Session，重新整理頁面後會消失。
